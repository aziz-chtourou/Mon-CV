Compétences techniques

Langages de programmation :

Java: Langage polyvalent utilisé pour le développement d'applications mobiles (Android), backend (JEE, Spring Boot) et logiciels.

Python: Langage interprété, populaire pour son utilisation en intelligence artificielle, data science, développement web (Django), et scripts automatisés.

C: Langage bas-niveau souvent utilisé pour le développement de systèmes embarqués, systèmes d'exploitation et logiciels nécessitant des performances élevées.

SQL: Langage standardisé pour gérer et interroger les bases de données relationnelles.

Développement web :

HTML5: Langage de balisage utilisé pour structurer le contenu des pages web.

CSS3: Langage de style utilisé pour définir la présentation et la mise en page des sites web.

PHP: Langage côté serveur utilisé pour développer des sites web dynamiques et gérer des bases de données.

JavaScript: Langage de script pour rendre les sites web interactifs et dynamiques.

Frameworks et bibliothèques JavaScript :

React.js: Bibliothèque JavaScript pour créer des interfaces utilisateur dynamiques et réactives.

Next.js: Framework basé sur React.js, optimisé pour le rendu côté serveur (SSR) et les performances web.

Node.js: Environnement d'exécution JavaScript pour le développement backend, permettant de gérer des applications serveur rapides et scalables.

Frameworks backend :

Django: Framework Python pour développer rapidement des applications web robustes et sécurisées.

JEE (Java Enterprise Edition): Plateforme Java pour créer des applications web et d'entreprise complexes et scalables.

Spring Boot: Framework Java permettant de créer des applications backend légères et faciles à configurer.

Frameworks front-end :

Angular: Framework JavaScript/TypeScript pour développer des applications web à page unique (SPA).

Bases de données :

MongoDB: Base de données NoSQL orientée documents, adaptée pour les applications nécessitant une grande flexibilité et scalabilité.
